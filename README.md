# Run MockUpGen locally

MockUpGen is a phone case template builder for creating, previewing, and exporting mockup templates.

No external API keys are required for the current feature set.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open `http://localhost:3000`
